# 한글 블록 분해기

한글 블록 분해기는 한글 음절 블록을 자모 단위로 풀어서, 결과를 읽기/문자 설명/복사까지 빠르게 처리할 수 있게 해 주는 NVDA 추가 기능입니다.

## 주요 기능

- 선택한 한글이 있으면 그 범위를 우선으로 분해합니다.
- 선택이 없을 때는 설정한 기본 범위(커서 아래 한 글자/현재 단어/현재 줄)로 분해합니다.
- 한글 문자만 입력/처리하며, 한글이 아닌 문자는 자동으로 제외됩니다.
- 겹글자 분해 옵션을 지원합니다 (`ㅘ -> ㅗㅏ`, `ㄳ -> ㄱㅅ`).
- 분해 결과 글자 사이 공백 삽입 옵션을 지원합니다.
- 입력/결과를 분리한 분해기 대화상자를 제공합니다.
- 분해기 대화상자는 `Escape` 키로 바로 닫을 수 있습니다.
- 제스처를 쓰지 않아도 도구 메뉴에서 분해기를 열 수 있습니다.
- 모든 명령을 입력 제스처에서 다시 연결할 수 있습니다.

## 기본 제스처

- `NVDA+alt+h`: 선택한 한글(선택이 없으면 설정한 기본 범위)을 분해해 문자 설명으로 읽기. 두 번 누르면 분해 결과를 클립보드 복사.
- `NVDA+ctrl+h`: 선택한 한글(선택이 없으면 설정한 기본 범위)을 분해해 클립보드에 복사.
- `NVDA+shift+h`: 분해기 대화상자 열기.

## 제스처 재할당

NVDA 메뉴 -> 환경설정 -> 입력 제스처에서 `한글 블록 분해기` 범주를 찾으면 됩니다.

기본 제스처가 없는 명령도 여기서 직접 연결할 수 있습니다.

- 겹글자 분해 켜기/끄기
- 글자 사이 공백 삽입 켜기/끄기
- 대화상자 실시간 갱신 켜기/끄기
- 기본 분해 범위 순환 전환(한 글자/단어/줄)

## 추가 기능 설정

NVDA 메뉴 -> 환경설정 -> 설정 -> `한글 블록 분해기`에서 기본 동작을 바꿀 수 있습니다.

- 겹글자 분해 기본값
- 공백 삽입 기본값
- 대화상자 실시간 갱신 기본값
- 텍스트 미선택 시 기본 분해 범위
